CDPL_NEXT PROJECT DOCUMENTATION
================================

PROJECT OVERVIEW
----------------
A high-fidelity defense-oriented web application built using Next.js and React. The project follows a "Technical HUD" aesthetic for navigation and incorporates advanced design elements like text scrambling (HackerLink), glassmorphism, and custom animations.

CORE TECHNICAL STACK
--------------------
- Framework: Next.js 16.1.6
- Library: React 19.2.3
- Language: JavaScript / TypeScript (Hybrid)
- Styling: Vanilla CSS (globals.css + Styled-JSX)
- Environment: Node.js (V20+ recommended)

PROJECT STRUCTURE
-----------------
/app
  Contains the App Router page definitions. Each folder represents a route.
  - /about: Company overview and leadership details.
  - /contact: Inquiries and contact forms.
  - /mas: Major Aerospace Systems division page.
  - /mgs: Major Ground Systems division page.
  - /mms: Major Marine Systems division page.
  - /products: Detailed product listings and specifications.
  - /resources: Technical documents, brochures, and media.
  - layout.js: Root layout containing the Navbar and Footer.
  - page.tsx: Main landing page implementation.

/components
  Reusable UI components.
  - Navbar.js: The core navigation component. Features a "Technical HUD" for mobile and a multi-column mega-menu for desktop. Includes the `HackerLink` scrambling effect.
  - Hero.js: Dynamic landing section for the homepage.
  - ProductCategories.js: Division-level navigation (MAS, MGS, MMS).
  - Products.js: Interactive product carousel and detail cards.
  - About.js: Informational sections for the About page.
  - Footer.js: Site-wide footer with branding and links.
  - Contact.js: Contact interface.
  - Resources.js: Resource listing and download interface.
  - Platform.js: Technical platform overview.

/public
  Static assets.
  - /partners: Icons and logos for CDPL and its divisions (masicon, cdpliconbl, etc.).

/styles
  Global styling.
  - globals.css: Defines the design system (color tokens, typography, hub-animations, and global resets).

KEY FEATURES
------------
- Technical HUD: A mobile-first navigation overlay with system status indicators, session IDs, and monospace typography.
- HackerLink Effect: JavaScript-driven text scrambling on link hover for a "cypher" aesthetic.
- Glassmorphism & Micro-animations: Heavy use of backdrop filters and pulsing dots for a premium feel.
- Divisional Separation: Clear branding hierarchy for Aerospace (MAS), Ground (MGS), and Marine (MMS) systems.

BUILD & DEVELOPMENT
-------------------
- npm run dev: Starts the development server.
- npm run build: Generates the production bundle.
- npm run start: Starts the production server.
- npm run lint: Runs ESLint checks.
